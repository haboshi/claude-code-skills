## 出力SVG仕様

| 項目           | 値                                                           |
| -------------- | ------------------------------------------------------------ |
| ViewBox        | 0 0 {width} {height}（デフォルト: 0 0 1920 1080）           |
| Width          | カスタマイズ可能（デフォルト: 1920px）                       |
| Height         | カスタマイズ可能（デフォルト: 1080px）                       |
| 日本語フォント | 'Noto Sans JP', 'Hiragino Sans', 'Yu Gothic', sans-serif    |
| 英語フォント   | 'Inter', 'Helvetica Neue', Arial, sans-serif                 |
| フォントスケール | `Math.min(width/1920, height/1080)` で自動調整             |

## カスタムテーマJSON形式

```json
{
  "name": "表示名",
  "type": "v3",
  "bg": ["#色1", "#色2", "#色3"],
  "accent": ["#色1", "#色2", "#色3"],
  "text": "#テキスト色",
  "subText": "#サブテキスト色"
}
```

### type の選択肢

| type           | 背景スタイル               |
| -------------- | -------------------------- |
| `v3`           | グラデーション+グリッド+ノイズ+グロー |
| `v2-neon`      | ラジアルグラデーション+グリッド+グロー |
| `v2-glass`     | グラデーション+半透明球     |
| `v2-geometric` | 幾何学分割+装飾             |


## タイトル分割ロジック

- 16文字以下: 1行表示
- 16文字超: `Intl.Segmenter` で単語境界を解析し、中央付近で2行に分割
- 日本語文字を検出した場合は `ja-JP` ロケール、それ以外は `en` ロケール
- 数字+助数詞（年、月、日、版 等）の結合に対応
