## 設計思想

学術調査の最大の課題は**ペイウォール**と**ハルシネーション**。

### ペイウォール対策

| アクセスレベル | 対応 |
|-------------|------|
| 全文公開（arXiv, open access） | WebFetch で全文取得 |
| アブストラクトのみ公開 | アブストラクトで要約、「全文未確認」と明記 |
| 完全非公開 | タイトル・著者のみ記録、内容は記述しない |

### ハルシネーション防止

学術論文のハルシネーションは特に危険（架空のDOI、架空の著者名、存在しないジャーナル）。

防止策:
1. WebSearch の検索結果に表示されたタイトル・著者をそのまま使う（推測しない）
2. DOI や arXiv ID が取得できた場合のみ引用する
3. 「この論文は検索結果に表示されたが、全文は確認していない」と正直に記述
4. 被引用数や出版年は、検索結果に明示されている場合のみ記載

## 推奨検索パターン

```
# 総説論文の発見
"{topic} survey review 2024 OR 2025"

# 特定手法の論文
"{method} {application} site:arxiv.org"

# 高被引用論文
"{topic} highly cited landmark paper"

# 最新動向
"{topic} state of the art SOTA 2025"
```
